# 70 爬楼梯

## 题目描述

假设你正在爬楼梯。需要 *n* 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**注意**：给定n是一个正整数

**示例 1：**

- 输入： 2
- 输出： 2
- 解释： 有两种方法可以爬到楼顶。
  1. 1 阶 + 1 阶
  2. 2 阶

**示例 2：**

- 输入： 3
- 输出： 3
- 解释： 有三种方法可以爬到楼顶。
  1. 1 阶 + 1 阶 + 1 阶
  2. 1 阶 + 2 阶
  3. 2 阶 + 1 阶

## 代码

```python
class Solution:
    '''
    k(n) = k(n-1)+k(n-2)
    '''
    def climbStairs(self, n: int) -> int:
        k = list(range(n + 1))
        for i in range(3, n + 1):
            k[i] = k[i - 1] + k[i - 2]
        return k[n]
```
